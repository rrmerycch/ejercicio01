
# ¿Qué es SEO?

SEO significa:

```txt
Search Engine Optimization
```

En español:

```txt
Optimización para Motores de Búsqueda
```

SEO son técnicas usadas para que una página web aparezca en mejores posiciones en Google.

---

# ¿Qué hace Google?

Google analiza páginas web y muestra resultados relacionados con lo que una persona busca.

## Ejemplo

Si alguien escribe:

```txt
cómo hacer pizza
```

Google buscará páginas relacionadas con pizza.

El SEO ayuda a que tu página tenga más posibilidades de aparecer primero.

---

# ¿Por qué es importante el SEO?

El SEO ayuda a:

- conseguir visitas
- mejorar visibilidad
- crecer en internet
- atraer clientes
- generar confianza

---

# Palabras clave

Las palabras clave son términos que las personas buscan en Google.

## Ejemplos

```txt
curso de programación
```

```txt
ropa deportiva
```

```txt
gatos pequeños
```

---

# ¿Cómo usar palabras clave?

Si tu página habla sobre gatos, puedes usar:

```txt
comida para gatos
cuidados de gatos
gatos pequeños
```

Las palabras deben verse naturales dentro del texto.

---

# Contenido

El contenido es toda la información de una página.

## Puede incluir:

- texto
- imágenes
- videos
- títulos
- listas

---

# ¿Qué contenido le gusta a Google?

Google valora contenido:

- útil
- claro
- original
- fácil de leer

---

# Encabezados HTML

Los encabezados organizan el contenido.

## Ejemplo

```html
<h1>Título principal</h1>

<h2>Subtítulo</h2>

<h3>Tema secundario</h3>
```

---

# ¿Qué es H1?

El `<h1>` es el título principal de la página.

## Ejemplo

```html
<h1>Curso SEO para Principiantes</h1>
```

Debe usarse una sola vez.

---

# Meta Description

Es el texto que aparece debajo del título en Google.

## Ejemplo

```html
<meta name="description" content="Aprende SEO desde cero">
```

---

# Título SEO

Es el título que aparece en Google.

## Ejemplo

```html
<title>SEO para Principiantes</title>
```

Debe ser:

- corto
- claro
- relacionado al tema

---

# URLs

La URL es el enlace de la página.

## Buena URL

```txt
misitio.com/curso-seo
```

## Mala URL

```txt
misitio.com/pagina?id=8392
```

---

# Imágenes y SEO

Las imágenes deben:

- cargar rápido
- tener nombres claros
- usar atributo ALT

## Ejemplo

```html
<img src="gato.jpg" alt="Gato pequeño">
```

---

# Velocidad de carga

Es el tiempo que tarda una página en abrir.

## Para mejorar velocidad:

- optimizar imágenes
- usar menos archivos pesados
- escribir código limpio

---

# Diseño Responsive

Responsive significa que la página funciona bien en:

- celulares
- tablets
- computadoras

---

# Tipos de SEO

## SEO On Page

Optimización dentro de la página.

### Ejemplos

- títulos
- contenido
- imágenes
- palabras clave

---

## SEO Off Page

Optimización fuera de la página.

### Ejemplos

- enlaces externos
- redes sociales
- recomendaciones

---

## SEO Técnico

Parte técnica del sitio web.

### Incluye

- velocidad
- seguridad
- estructura
- indexación

---

# Backlinks

Son enlaces desde otras páginas hacia tu sitio.

Google interpreta eso como confianza.

---

# Buenas prácticas SEO

- escribir contenido original
- usar títulos claros
- optimizar imágenes
- usar palabras clave naturalmente
- mejorar velocidad

---

# Errores comunes

## Copiar contenido

Google penaliza contenido duplicado.

---

## Usar demasiadas palabras clave

Puede verse como spam.

---

## Página lenta

Las personas abandonan páginas lentas.

---

```html
<!DOCTYPE html>

```
